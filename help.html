Mining Guide

Step-by-step guide to start mining SHIVRAI HON on your Windows PC. No technical knowledge needed!

1
Download the Miner

Go to the download page and get the Windows Miner ZIP file:

👉 Click here to download SHIVRAI HON Windows Miner v2.0

Click on SHIVRAI-HON-Windows-Miner-v2.0.zip to download. The file is about 19 MB.

✅ Tip: Use Google Chrome or Microsoft Edge for the fastest download.
2
Extract the ZIP File

After downloading, follow these steps:

1. Open your Downloads folder

2. Find SHIVRAI-HON-Windows-Miner-v2.0.zip

3. Right-click on the file → Click "Extract All..."

4. Choose a location like C:\SHIVRAI-HON\

5. Click "Extract"

✅ Tip: Make sure all files are in one folder. You should see bitcoind.exe, bitcoin-cli.exe, 2 BAT files, and many DLL files.
3
Handle Windows Security Warning

Windows may show a security warning because the software is new. This is normal and safe.

If you see "Windows protected your PC":

1. Click "More info"

2. Click "Run anyway"

⚠️ Note: If your antivirus blocks bitcoind.exe, add an exception for your SHIVRAI-HON folder. The software is open source and safe — you can verify the code on GitHub.
4
Start the Node

In your SHIVRAI-HON folder:

1. Find 1_START_NODE.bat

2. Double-click on it

3. A black window (Command Prompt) will open

4. The program will automatically:

   ✅ Create a data folder at %APPDATA%\SHIVRAI-HON

   ✅ Create configuration file (bitcoin.conf)

   ✅ Connect to seed node (bore.pub:34969)

   ✅ Start syncing with the network

✅ Tip: Keep this window open! Do NOT close it. The node must be running for mining to work.
5
Open the Control Panel

Open a NEW window (do not close the first one!):

1. Go to your SHIVRAI-HON folder again

2. Find 2_CONTROL_PANEL.bat

3. Double-click on it

4. You will see a menu with options [0] to [9]

6
Create Your Mining Address

In the Control Panel:

1. Type 3 and press Enter

2. A new address will appear (starts with bc1...)

3. Copy this address — select it and press Ctrl+C

4. Press any key to go back to the menu

⚠️ Important: Save this address somewhere safe! This is YOUR wallet address where your mined HON will go.
7
Start Mining! 🎉

In the Control Panel:

1. Type 4 and press Enter

2. It will ask for your mining address

3. Paste the address you copied (right-click → Paste)

4. Press Enter

5. Mining starts! 🎉

✅ Congratulations! You are now mining SHIVRAI HON! Every block you mine earns you 2000 HON. Check your balance anytime by pressing [2] in the Control Panel.
8
Check Your Balance

To see how many HON you have mined:

1. Open the Control Panel (if not already open)

2. Type 2 and press Enter

3. Your balance will be shown

✅ Note: Mined coins become available after 100 blocks. So if you just started mining, your balance may show 0 for a while. Keep mining!

Frequently Asked Questions

What is SHIVRAI HON? / SHIVRAI HON काय आहे? +
SHIVRAI HON (HON) is an independent cryptocurrency built on Bitcoin Core. It uses SHA-256 Proof of Work mining with 2000 HON reward per block and 5-minute block time.

SHIVRAI HON (HON) हे Bitcoin Core वर बनवलेले स्वतंत्र cryptocurrency आहे. यात SHA-256 Proof of Work mining वापरले जाते, प्रत्येक block ला 2000 HON reward आणि 5 मिनिटे block time आहे.
Is it free to mine? / Mining मोफत आहे का? +
Yes! Mining is completely free. You just need a Windows PC and internet connection. Download the miner, run it, and start earning HON.

हो! Mining पूर्णपणे मोफत आहे. तुम्हाला फक्त Windows PC आणि इंटरनेट कनेक्शन लागते. Miner डाउनलोड करा, चालवा, आणि HON कमवायला सुरू करा.
Why is my balance showing 0? / माझा balance 0 का दिसतो? +
Mined coins need 100 block confirmations before they appear in your balance. Keep mining — after 100 blocks, your rewards will show up!

Mine केलेले coins balance मध्ये दिसण्यासाठी 100 block confirmations लागतात. Mining चालू ठेवा — 100 blocks नंतर तुमचे rewards दिसतील!
Can I mine on any computer? / कोणत्याही computer वर mine करता येते? +
Currently, the miner works on Windows PCs. Any Windows 10/11 computer with an internet connection can mine SHIVRAI HON.

सध्या miner Windows PC वर काम करतो. कोणताही Windows 10/11 computer इंटरनेट कनेक्शन सह SHIVRAI HON mine करू शकतो.
Is it safe? / हे सुरक्षित आहे का? +
Yes! SHIVRAI HON is built on Bitcoin Core — the most secure blockchain codebase. The source code is fully open source on GitHub. You can verify everything yourself.

हो! SHIVRAI HON हे Bitcoin Core वर बनवलेले आहे — सर्वात सुरक्षित blockchain. Source code GitHub वर पूर्णपणे open source आहे.
Node not connecting? / Node connect होत नाही? +
Make sure the seed node is active. If you see 0 peers after a few minutes, the seed node may be temporarily offline. Try again later or contact us on Telegram.

Seed node active असल्याची खात्री करा. काही मिनिटांनंतर 0 peers दिसत असल्यास, seed node तात्पुरता offline असू शकतो. नंतर पुन्हा प्रयत्न करा.
