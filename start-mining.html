Start Mining SHIVRAI HON

Mine HON on your computer. No special hardware needed. Choose your method below — simple BAT files or the full Desktop App.

Block Reward
2000 HON
Block Time
5 Minutes
Algorithm
HONPoW
Rewards
Permanent
1
Download the Miner
⬇ Download SHIVRAI HON Miner v2.1 Secure (19MB)

Click on SHIVRAI-HON-Windows-Miner-v2.1-Secure.zip to download.

2
Extract the ZIP

1. Open your Downloads folder

2. Right-click on the ZIP → "Extract All..."

3. Choose location like C:\SHIVRAI-HON\

4. Click "Extract"

✅ You should see: bitcoind.exe, bitcoin-cli.exe, 2 BAT files, many DLL files, and README.txt
3
Handle Windows Security

Windows may show "Windows protected your PC" — this is normal.

1. Click "More info"

2. Click "Run anyway"

⚠️ If antivirus blocks bitcoind.exe, add your SHIVRAI-HON folder as an exception. The code is open source — verify on GitHub.
4
Start the Node

Double-click 1_START_NODE.bat

A black window opens — the node automatically:

✅ Creates data folder   ✅ Generates secure password   ✅ Connects to network

✅ Keep this window open! Do NOT close it.
5
Open Control Panel

Open a NEW window — double-click 2_CONTROL_PANEL.bat

You'll see a menu with options [1] to [9]

6
Create Mining Address

Type 3 → Press Enter → New address appears (bc1q...)

Copy this address — this is YOUR wallet!

⚠️ Save this address somewhere safe!
7
Start Mining! 🎉

Type 4 → Paste your address → Press Enter

Mining starts! Every block = 2000 HON

✅ Check balance anytime: Press [2] in Control Panel
✅ Mined coins appear after 100 block confirmations

Frequently Asked Questions

What is SHIVRAI HON? +
SHIVRAI HON (HON) is an independent decentralized cryptocurrency. It uses the HONPoW mining algorithm with 2000 HON reward per block and 5-minute block time. Named after Chhatrapati Shivaji Maharaj's historic gold coin.
Is mining free? +
Yes! Mining is completely free. You just need a computer and internet connection. No special hardware required — any CPU can mine SHIVRAI HON.
Why is my balance showing 0? +
Mined coins need 100 block confirmations before they appear in your balance. Keep mining — after 100 blocks, your rewards will show up!
What's the difference between Simple Mining and Desktop App? +
Simple Mining uses BAT files in Command Prompt — quick and easy to start. The Desktop App gives you a beautiful dashboard with balance charts, mining controls, transaction history, and send/transfer features.
Is it safe? +
Yes! SHIVRAI HON is built on proven cryptographic technology. Each install generates a unique secure password. The source code is fully open source on GitHub — verify everything yourself.
How much HON can I earn? +
Every block mined earns you 2000 HON. With a 5-minute block target, you can potentially earn ~576,000 HON per day if you're the only miner. Rewards are permanent — no halving!
Node not connecting? +
Make sure the seed node (bore.pub:34969) is active. If you see 0 peers after a few minutes, the seed node may be temporarily offline. Check back later or join our community for support.
